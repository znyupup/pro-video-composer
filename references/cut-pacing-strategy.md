# 节奏分段策略

不要全程 1-2s/镜(累 + 信息过载)也不要全程 8s+(慢 + 走神)。**分段节奏**最佳。

## 大 V 节奏曲线的真实结构

回看影视飓风/金融视频原始数据:
- **平均** 1.26-1.76s/镜 — 这是**全片平均**,不是全程目标
- **最长** 3.8s(金融) / 7.5s(影视飓风) — 主讲段长镜头
- 节奏曲线:**开场快(Hook 1-2s/镜) → 讲解稳(3-8s/镜) → 转折回快(1-2s/镜) → 收尾稳**

## 分段建议

| 段位 | 镜头时长 | 切镜密度 | 视觉策略 |
|---|---|---|---|
| **开场 Hook**(0-30s) | 1-2s/镜 | 极快 | 信息轰炸 + 痛点 + 大 V 节奏 |
| **主讲段**(30s-80%) | **3-8s/镜** | 稳 | 单镜头讲完一概念,长留白让观众消化 |
| **转折/对比段** | 1-2s/镜 | 短爆发 | 强化 contrast,突出转折点 |
| **收尾 + CTA** | 2-5s/镜 | 稳 + 收 | 沉淀情绪 + CTA |

## 长镜头的设计要素(主讲段 3-8s/镜)

3-8s 单镜头不会让观众觉得慢,前提是**镜头内有内在节奏**:

1. **画面变化**:zoom in/out、pan、subtle motion(数字人 PiP 持续说话也是变化)
2. **UI 动效持续**:bg gradient 漂移、装饰元素脉动、光晕呼吸
3. **关键词高亮节奏**:口播说到关键词时 callout 弹出,镜头不切但视觉点持续刷新
4. **B-roll 叠加**:主层不动,但 PiP / 角标 / 顶层小卡片轮播

**反例**:5s 镜头里 zero motion + zero UI 变化 → 观众秒走

## 应用模式

### Hook 段(1-2s/镜)
- 11-12 个 cut 平均 1.78s/镜(影视飓风同款)
- 每 cut 一个完整视觉信息(标题/对比/演示/数据)
- 必须含"震撼成片镜头"(开场惊艳)

### 主讲段(3-8s/镜)
- 真人入画讲解段:3-5s/镜(章节切换才硬切)
- Demo 录屏段:4-8s/镜(操作有内在节奏)
- AI 生成动画穿插:1-2s/镜(短爆发,跟前后段对比)

### 转折段(1-2s/镜)
- 概念对比:before / after 各 1.5s
- 数据冲击:数字 zoom in 0.8s + 快闪 1.2s

### 收尾(2-5s/镜)
- CTA 大字 3s
- 主持人收尾镜头 4-5s
- 头尾各 0.5s 黑场

## 反模式

- ❌ 全程 1-2s/镜:观众疲劳,大脑过载
- ❌ 全程 8-15s/镜:走神,完播率掉
- ❌ 长镜头无 motion:必走神
- ❌ 收尾不稳镜头:情绪断,CTA 接不住

## Checklist

- [ ] Hook 段 1-2s/镜 ✓ 大 V 节奏
- [ ] 主讲段 3-8s/镜 + 内置动效持续
- [ ] 转折段 1-2s/镜 短爆发
- [ ] 收尾段 2-5s/镜 + CTA 沉淀
- [ ] 全片有"震撼成片镜头"(高光时刻)
- [ ] 头尾各 0.5s 黑场
